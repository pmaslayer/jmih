Welcome to Scrolly Point Example

This page demonstrates a scrolly point effect similar to the one on the provided example.

The black dot will move around the screen as you scroll, creating a dynamic visual effect.

How it works:

  • The dot is positioned absolutely in the center of the viewport.
  • As you scroll, the dot moves along with the scrolling element.
  • The movement is smooth thanks to CSS transitions.
